# Astro Library Frontend

React, TypeScript, Vite, and Tailwind CSS frontend for Astro Library.

## Setup

```bash
npm ci
npm run dev
```

The development server proxies `/api` to the backend at `http://127.0.0.1:8000`.

## Checks

```bash
npm run format:check
npm run lint
npm run typecheck
npm run build
```
